## Summary
This repository provides end-to-end infrastructure automation for deploying the TrainTicket microservice benchmark on Kubernetes (especially in CloudLab), including cluster bootstrap scripts, manifest tooling, deployment helpers, and tracing access workflows (SkyWalking UI/GraphQL). It is designed to make repeatable systems experiments easier by standardizing setup, service rollout, and experiment readiness checks.
