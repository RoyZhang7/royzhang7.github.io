## Summary
This project is a fork of TrainTicket auto-query tooling with a concurrent load generator and warmup mode to populate database state before experiments. It also includes practical scripts for building updated service images and redeploying pods in Kubernetes, making it useful for high-throughput workload generation and controlled performance testing.
